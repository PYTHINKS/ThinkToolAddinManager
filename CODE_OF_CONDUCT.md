# Code of Conduct

## Our Pledge

We want this project to be a welcoming and productive space for everyone. Contributors are expected to be respectful, constructive, and considerate when opening issues, reviewing code, or participating in discussions.

## Unacceptable Behavior

Unacceptable behavior includes harassment, discriminatory language, personal attacks, sustained disruption, or publishing private information without permission.

## Enforcement

Project maintainers may remove comments, close issues, or restrict participation when behavior is harmful to the project or community.

## Reporting

Report conduct concerns privately to the repository owner. Include relevant links or screenshots when possible.
